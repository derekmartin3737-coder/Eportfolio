# Artifact Organization

This folder is intentionally organized around the assignment deliverables:

- `resume-cover-letter/`
- `design-proposal/`
- `self-learning-assignment/`
- `final-team-report/`
- `team-charter/`
- `reflection-essay/`

Recommended naming style:

- `[your-name]-resume.pdf`
- `[your-name]-cover-letter.pdf`
- `capstone-design-proposal-final.pdf`
- `self-learning-assignment.pdf`
- `final-team-report.pdf`
- `team-charter-final.pdf`
- `reflection-essay.pdf`

This repo does not include invented document content. Add your real files when they are ready.
